# Monster
最简单的刷怪插件，怪物沿自定义线条移动，被打死后重生


> 版本：1.0 | 适用于：虚幻引擎 5.0+ | 作者：fuwenju

---

## 1. 概述

Monster Beta 是一款简单易用的刷怪插件，专为需要快速实现怪物生成与重生逻辑的游戏项目设计。它提供了开箱即用的怪物 AI 移动模式和重生系统，无需编写复杂代码，即可在几分钟内为你的场景添加完整的刷怪功能。

无论是原型开发、快速迭代，还是最终项目中的怪物管理，Monster Beta 都能帮你节省大量时间。

---

## 2. 功能特性

- **一键刷怪**：将 Actor 拖入场景，配置样条线即可开始生成。
- **自定义线条移动**：怪物可沿自定义路径（线条/样条线）巡逻移动，支持循环与往返模式。
- **自动重生**：怪物被击杀后，在指定延迟后自动在刷怪点重生，保持关卡挑战性。
- **纯蓝图实现**：全部逻辑基于蓝图，无需 C++ 编译，即插即用，方便二次修改。
- **轻量高效**：对性能影响极小，适合大规模怪物部署。
- **高度可配置**：每个刷怪点和怪物实例都暴露了丰富的参数，方便按需调整。

## 4. 安装与启用

### 4.1 从 Fab 安装（推荐）

1. 在 Epic Games 启动器中，进入 Fab 库 (Fab Library)。
2. 找到 Monster Beta，点击 **安装到引擎 (Install to Engine)**。
3. 选择目标 UE 版本，等待安装完成。

### 4.2 手动安装

1. 将 MonsterBeta 插件文件夹复制到：
   - 项目专用：`[项目根目录]/Plugins/`
   - 全局使用：`[UE安装目录]/Engine/Plugins/`
2. 打开 UE 项目，进入 **编辑 (Edit) > 插件 (Plugins)**。
3. 在列表中找到 Monster Beta，勾选 **启用 (Enabled)**。
4. 重启编辑器使插件生效。

**验证安装**：启用后，内容浏览器中出现 `MonsterBeta` 目录即表示安装成功。

## 5. 快速上手（5 分钟入门）

### 步骤 1：创建刷怪点

1. 在内容浏览器中，导航至 `MonsterBeta/Blueprints/`。
2. 将 `BP_Line_Monster` 拖入场景。
3. 在细节面板中，设置怪物模型。

### 步骤 2：设置移动路径

1. 选中场景中的怪物，设置样条线。
2. 选择 Patrol（巡逻模式）。
3. 在场景中使用 Spline Path 绘制巡逻路线，然后将样条线资产指定给怪物的 Patrol Path 属性。

### 步骤 3：运行游戏

点击 **运行 (Play)**，怪物将按照你设定的路径移动，击杀后会在一段时间后自动重生。

## 6. 常见问题

**Q1：插件启用后，内容浏览器中看不到 MonsterBeta 目录？**

A：在内容浏览器中点击 **设置 (Settings) > 显示插件内容 (Show Plugin Content)**。或者搜索 Monster 即可找到。

**Q2：支持多人游戏吗？**

A：当前版本为单机设计。若需多人网络同步，需要在怪物和刷怪点中添加 Replicated 变量和 RPC 函数进行适配。
